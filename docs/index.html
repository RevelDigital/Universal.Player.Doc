revel-universal-player documentation

RevelDigital Universal Player

Implemented Module Types

  • Clock - Analog or digital clock.

  • StaticImage - Single static image.

  • Gadget - Google Gadget.

  • Text - Static text.

  • RichText - Richly formatted text.

  • Image - Image slideshow with transitions.

  • Web - Web content.

  • Ticker - Scrolling marquee/ticker.

  • Shape - An image of a shape

  • QRCode - A QR code of text

  • HotSpot - Touch regions in kiosk applications.

  • TV - Live TV or video feed.

Examples

Listening For A Click

Controller.onClickEvent.subscribe(
    (event: MouseEvent) =>{
        console.log(event);
    });

Listening For A Command

Controller.onCommandCallback.subscribe(
    (command) =>{
        console.log(command);
    }
);
